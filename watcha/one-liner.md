# REDSecret - 一句话介绍

**用图片隐藏你的真心话——表面是数据，暗层是秘密。**

---

## 详细介绍

REDSecret 是一款基于 DCT 频域隐写技术的 AI 图片隐写工具，可以将秘密信息无损地嵌入到 JPEG 图片中，肉眼完全无法察觉。

### 核心功能

**1. 图片隐写**
基于 DCT（离散余弦变换）频域隐写算法，将任意文本信息嵌入到 JPEG 图片的频率系数中。嵌入后的图片在视觉上与原图完全一致，但只有知道密钥的人才能提取隐藏信息。

**2. 真心话图卡（Secret Truth Card）**
生成精美的图片卡片：表面展示公开数据（如股票分析、天气预报），暗层则隐藏你的真心话。发送者在社交媒体上发布一张"正常"的图卡，只有收件人才能读取隐藏的秘密信息。

**3. 双层叙事**
- **表层（Surface）**：公开可见的内容，如数据报告、祝福卡片、日常图片
- **暗层（Hidden）**：只有发送者和接收者知道的秘密信息

**4. 隐私优先**
不需要任何第三方服务器，图片本身即加密容器。只有拥有正确密钥的人才能提取信息。

### 使用场景

- 通过社交媒体发送私密信息
- 情书与告白：在一张看似普通的图片中藏入爱的告白
- 机密数据传输：用图片传递敏感文本信息
- 彩蛋：在公开发布的内容中埋下隐藏彩蛋

### 技术亮点

- Python PIL/Pillow + DCT 数学实现，无需深度学习
- 开源项目，GitHub 完全公开
- 支持 Web 在线体验
- 轻量级，易于部署和二次开发
