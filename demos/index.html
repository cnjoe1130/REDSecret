三种隐写方式

选择最适合你场景的方案

像素隐写

QIM 加密算法，将信息藏在图片像素中。肉眼完全不可见，适合 1v1 文件传输。

QR 角标隐写

加密二维码贴在图片角落，Reed-Solomon 纠错，抗压缩抗损坏。

零宽字符隐写

Unicode 零宽字符编码，藏在文字里，肉眼完全不可见。

安装

一行命令，支持全平台。

$ pip install Pillow numpy qrcode
02

加密

输入密文 + 密钥，生成隐写图。

$ python encrypt.py "密文" --key "key"
03

分享

发图给对方，告诉 ta 密钥。

$ python decrypt.py img.png "key"
